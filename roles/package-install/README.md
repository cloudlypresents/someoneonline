# Task 1 - Install Packages
## Beschreibung
Die Rolle installiert die angegebenen Packages dem Remote-Server.

Die zu installierenden Packages werden in der Datei
`files/package_requirements.txt` zeilenweise hinterlegt.

#### Beispiel
```
python3
python3-dev
python3-pip
git
```
